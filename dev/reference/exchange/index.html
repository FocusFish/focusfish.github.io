    Exchange

    The Exchange module facilitates data movement in and out of UVMS by defining one interface that plugins interact with. Plugin examples range from receiving AIS data to sending email.

    • Exchange Module
    • Exchange

    Artifacts

    Exchange Module

    <dependency>
    <groupId>fish.focus.uvms.exchange</groupId>
    <artifactId>exchange-module</artifactId>
    <version>5.3.32</version>
</dependency>

    implementation 'fish.focus.uvms.exchange:exchange-module:5.3.32'

    Exchange Client

    <dependency>
